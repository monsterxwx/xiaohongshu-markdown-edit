# ✨ 小红书无痛封面生成器 (Xiaohongshu Cover Generator)

一个极简、优雅、开箱即用的小红书风格图文封面生成工具。基于 Vue 3 打造，只需输入 Markdown 文本，即可一键渲染出拥有「高级设计感排版」的小红书标准 3:4 比例高清封面，并支持一键下载 3 倍超清大图。

> 告别繁琐的作图软件，让你专注于文字内容创作的辅助利器！

## 🌟 核心特性 (Features)

- **📝 双模式引擎 (Markdown排版 & 图片加字)**：
  - **文字排版**：左侧输入纯文本结构，加粗、标题、列表、引用……右侧即可全自动生成精美图文。
  - **图片加盖标题**：一键上传 3:4 背景封面图，自由添加浮空标题并支持鼠标**任意拖拽定位**。同时你可以精细调整**字体颜色、大小和行高**。所有视觉参数都会通过浏览器的缓存自动记忆，防刷新丢失！
- **💅 沉浸式 SaaS 控制台体验 (PC端)**：电脑端自动开启满屏应用级界面的沉浸模式，左侧独立固定控制台，右侧具备无限网格画板+阴影浮空质感，创作非常舒服。
- **📱 完美的移动端适配**：在手机上浏览也不会出现局促的滚动条！采用流式排版一境到底滚动，并且使用 CSS transform 将生成画布精准适配手机主流屏宽。
- **🎨 高级质感排版引擎**：
  - 自动引用高颜值开源字体库：**思源黑体 (Noto Sans SC)** 作骨架，**思源宋体 (Noto Serif SC)** 为金句引用点睛。
  - 苹果级极简暖白/米灰渐变底色，高级的弱阴影。
  - 特别加粗强化、标题自动添加高亮前缀块。
- **📸 3 倍超高清无损导出**：点击导出图片，不会截取被缩放的 DOM 屏幕显示尺寸，而是从底层强制以原始的 `1080x1440` 小红书完美比例，按原生 3 倍像素比 (`pixelRatio: 3`) 进行无损渲染导出，保障文字在任何设备上清晰度极佳。
- **🛠 内置快捷工具栏**：不知道 Markdown 语法也没关系？顶部内置了快捷排版加粗、大标题等工具栏菜单，支持选中文字后自动包边。

## 🚀 快速启动 (Getting Started)

本项目使用 [Vue 3](https://vuejs.org/) 开发。如果你准备在本地运行：

1. **安装项目依赖**

  ```bash
  # 如果你使用 pnpm (推荐)
  pnpm install
  
  # 若使用 npm
  npm install
  ```

2. **安装核心处理库**
  如果 `package.json` 没有预装这两个库，请手动添加：
  ```bash
  pnpm add marked html-to-image
  ```

3. **启动开发服务器**

  ```bash
  pnpm dev
  ```

4. **立刻使用：** 打开浏览器默认的 `http://localhost:5173` 即可开始排版你的第一张封面图片啦！

## 💻 技术栈推荐 (Tech Stack)

- **[Vue 3](https://v3.vuejs.org/)** - 灵活且强大的前端响应式框架 (Composition API setup 语法)
- **[marked](https://github.com/markedjs/marked)** - 高性能的核心 Markdown 编译引擎
- **[html-to-image](https://github.com/bubkoo/html-to-image)** - 用来截取 DOM 元素并原画质输出下载的救星

## ✍️ 使用小贴士
1. 在正文里把**最重要的两三个关键词**使用 `**加粗**` 括起来，它们会自动带有小红书主色调 (#ff2442) 以及微弱的高亮背景提示。
2. 尽量多使用 `> 金句提炼`，生成的侧边大红条和思源宋体组合简直不要太好看。
3. 使用 `H1大标题` 和 `H2小标题` 来梳理你的文字层级，生成器自动计算了最佳的段间距与字间距，不需你在编辑器里猛按回车排版！
