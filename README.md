# ✨ 自媒体创作者的「超级排版工厂」
*(Xiaohongshu Cover & WeChat Article Markdown Generator)*

一个极简、优雅、纯原生前端构建的多平台自媒体图文生成工具。基于 Vue 3 打造，集成**小红书无痛封面图生成**与**微信公众号长文沉浸式排版**两大核心引擎。

支持直接打通 DeepSeek AI 模型，写字只管写，格式甚至内容润色全部交给 AI 自动接管！无需注册，所有配置本地缓存记忆，安全且轻量。

## 🌟 核心特性 (Features)

### 🥑 一、小红书图文封面生成器 (Cover Engine)
- **文字与背景的双模融合**：一键上传 3:4 背景图，自由添加浮空标题并支持任意拖拽定位。
- **无脑式文本重构**：左侧纯文本结构，右侧直接呈现带“高级弱阴影”与“思源字体族”的图文封面。
- **3 倍超高清无损导出**：通过底层按原生 3 倍像素比 (`pixelRatio: 3`) 进行无损渲染导出 `1080x1440` 小红书完美比例封面图片，确保任何设备显示极致清晰。

### ✍️ 二、公众号高定长文排版引擎 (Article Engine)
- **真正的所见即所得**：左侧 Markdown 编辑，右侧手机壳模型毫秒级同步渲染真实观感效果。
- **5 套顶流自媒体美学主题零延迟切换**：
  - 💎 **科技职场风格**：专业严谨，带有强视觉引导的高亮卡片，适合行业干货。
  - 🍏 **苹果极简风格 (Minimal)**：克制的高级感，极具呼吸感的字号间距与留白设计。
  - 💌 **情感走心风格 (Emotion)**：暖调樱花粉、温和的圆角与引用框，带给读者倾诉的安全感。
  - 📘 **深度技术风格 (Indigo)**：深度长文标配，优雅的暗蓝色系与完整的技术大表单支持。
  - 💻 **极客开发风格 (Geek)**：带有 Mac 终端红黄绿顶栏风格的代码域、专属的行内代码胶囊保护。
- **纯前端安全复制协议**：渲染完毕后，点击【🔗 一键复制排版】，系统将采用原生流媒体协议将 `text/html` 的带样式富文本写入剪贴板，您只需在公众号后台直接 `Ctrl+V`，没有任何数据通过后台 API 传输，绝对隐私。

### 🤖 三、DeepSeek 引擎级整合 (AI Core)
无需复杂配置，点开右上角密钥输入框，绑定好个人的 DeepSeek API Key 即可唤醒左侧的智能辅助大军：
- **📐 一键排版**：你的内容全部连在一起了？点击 AI，瞬间帮你把文章加上优雅的 Markdown 骨架结构，不改变你的一字一句，只做物理结构调整。
- **✨ AI 润色**：输入简陋的大纲或者草稿，指令直达云端，返回一篇结构完美、措辞优美的高配长文。

## 🚀 快速启动 (Getting Started)

本项目使用 [Vue 3](https://vuejs.org/) 开发。

1. **安装项目依赖**
  ```bash
  # 推荐使用 pnpm
  pnpm install
  ```

2. **启动开发服务器**
  ```bash
  pnpm dev
  ```

3. **立刻使用**：默认访问 \`http://localhost:5173\`。所有功能即点即用，AI 配置和封面拖拽位置全持久化在您的本地 LocalStorage 中。

## 💻 核心技术栈 (Tech Stack)

- **[Vue 3](https://v3.vuejs.org/) + Vue Router** - 灵活且强大的前端响应式框架及多模式路由。
- **[marked.js](https://github.com/markedjs/marked)** - 构建强大自定义 Token 劫持解析能力的长文排版引擎核心。
- **[html-to-image](https://github.com/bubkoo/html-to-image)** - 用来截取 DOM 元素并原画质输出下载小红书保存图像的救星。
